# 台灣相關產業搜尋詞庫 - 2026-05-19

## 目的

本文件整理台灣使用者在 Google 搜尋 / Google Maps 可能會用來找「整復、推拿、整骨、結構調理、物理治療、徒手治療、運動按摩」相關服務的詞。

這不是要把全部詞都塞進 Google Business Profile。實際使用分四種：

- `P0 主攻`：官網與 GBP 都要安全承接。
- `P1 可承接`：官網 / FAQ / GBP 描述可自然出現。
- `P2 比較承接`：可寫在 FAQ 或比較型內容，不當作主服務名。
- `P3 只追蹤`：內部追蹤或廣告排除，不公開主打。

## 使用原則

- 見觀可以搶搜尋意圖，但不能把自己寫成醫療院所、物理治療所或治療服務。
- `治療`、`復健`、`矯正`、`根治`、疾病名與疼痛名，公開內容只能作為使用者問法、比較說明或服務邊界，不寫成見觀提供的療效。
- Google Maps / GBP 優先用安全服務詞：`傳統整復推拿`、`身體結構評估`、`身體結構觀察`、`結構調理`、`徒手調理`、`運動按摩`。
- 官網可以用 FAQ 承接高風險詞：例如「見觀不是物理治療所」、「結構治療這個詞常被怎麼理解」、「整復推拿和物理治療差在哪」。

## 搜尋意圖承接規則

本專案要做的是「使用者搜尋這些詞時，Google Search / Maps 能理解見觀和該需求相關」，不是把每個詞都改成見觀正式服務名稱。

可執行方式：

- 所有相關詞都可以放進內部詞庫、Search Console 追蹤、SERP 觀察與內容規劃。
- 安全服務詞可以放進 GBP 服務項目與商家描述，例如 `傳統整復推拿`、`身體結構評估`、`結構調理`、`徒手調理`、`運動按摩`、`肌筋膜放鬆`。
- 高風險詞可以放在 FAQ / 比較頁裡當使用者問法，例如 `物理治療`、`徒手治療`、`復健`、`結構治療`、`矯正`。
- 症狀詞可以用身體觀察筆記承接，例如 `肩頸卡`、`久坐腰痠`、`骨盆歪`、`高低肩`、`富貴包`。
- Maps 要靠 GBP 欄位、評論語意、照片說明、貼文與外部一致性一起累積，不是靠把所有詞塞進店名。

不可執行方式：

- 不把 `物理治療所`、`復健診所`、`治療中心` 寫成見觀身分。
- 不在 GBP 服務項目新增實際沒有提供的服務。
- 不用 `治療`、`根治`、`矯正成功`、`改善疾病` 這類結果承諾。
- 不做假 PTT / Dcard / 評論 / 排行榜。

## 地區組合規則

實際搜尋常不是單一詞，而是地區加服務詞。見觀可合法承接的地區以兩店與自然服務半徑為主。

優先地區：

- 台南
- 台南市
- 北區
- 台南北區
- 南區
- 台南南區
- 北安路
- 明興路

可追蹤但不要做假店點頁：

- 安平
- 安南區
- 永康
- 中西區
- 東區
- 仁德
- 歸仁
- 新市
- 善化

組合格式：

- `地區 + 服務`：台南整復推拿、北區整復、南區推拿
- `地區 + 服務 + 推薦`：台南整骨推薦、北區整復推拿推薦
- `地區 + 服務 + 費用`：台南整復費用、台南整骨費用
- `地區 + 身體狀態`：台南肩頸按摩、台南骨盆歪
- `服務 + 差別`：整復推拿物理治療差別、運動按摩徒手治療差別
- `服務 + PTT / Dcard`：台南整復推薦 PTT、台南物理治療 Dcard
- `服務 + 附近 / 營業中 / 評分最高`：整骨推薦評分最高、推拿附近
