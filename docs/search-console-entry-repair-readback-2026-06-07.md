1. 搜尋需求不是沒有，問題是入口沒有接好。
   `台南整骨` 824 曝光、`台南整復` 621 曝光、`整復` 737 曝光，都有量，但 CTR 偏低。

2. 高意圖推薦型詞是目前最值得修的 SERP 入口。
   `台南整復推薦` 320 曝光 / CTR 1.56% / 平均排名 8.93；`台南整骨推薦` 305 曝光 / CTR 0.33% / 平均排名 10.47；`台南整復推拿推薦` 166 曝光 / CTR 1.20% / 平均排名 8.28。

3. 舊 PHP / 舊架構 URL 仍大量佔據搜尋入口。
   `paper/other_select_index.php?id=3718...` 有 2,271 曝光 / 85 點擊；`paper/other_page.php?id=11272` 有 1,230 曝光 / 37 點擊。這代表舊 URL 轉移與索引清理仍是 P0，不只是寫新內容。

4. `/services/tainan-tuina/` 尚未進入本次 GSC 網頁表。
   服務頁仍不能被當成已接住搜尋需求。下一步應先查 URL 檢查、內鏈與 sitemap，而不是直接大改全文。

5. 到店與費用頁有曝光但 CTR 弱。
   `/visit/?store=south` 665 曝光 / CTR 1.20%；`/visit/?store=north` 614 曝光 / CTR 0.81%；`/south/pricing/` 181 曝光 / CTR 0%；`/north/pricing/` 285 曝光 / CTR 1.75%。
