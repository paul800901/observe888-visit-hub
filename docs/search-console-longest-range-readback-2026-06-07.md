## 判讀

1. 最長期間總量已更新為 2,233 點擊 / 68,377 曝光，這是目前本機 zip 可驗證的基準。
2. 最高曝光查詢仍是泛詞 `整復`，但 CTR 只有 0.81%，代表搜尋入口還有摘要與承接頁改善空間。
3. 品牌查詢點擊最高，`見觀結構調理整復所` 與 `見觀` 合計貢獻大量高 CTR 點擊。
4. 非品牌核心詞中，`台南整復` 點擊最高，`台南整骨` 與 `整復` 曝光高但 CTR 偏低。
5. 首頁仍是最大現代 URL 承接頁；舊 `.php` / 舊架構 URL 的頁面曝光列加總仍高於現代 URL，技術轉址與索引清理不可跳過。
6. 本資料只證明 2026-06-07 zip 匯出的 Search Console 後台數字，不代表公開排名、Maps 排名或索引狀態已改善。
