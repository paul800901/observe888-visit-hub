# Search Console x Google SERP 補充讀回 - 2026-05-20

## 用途

這份文件把 2026-05-20 使用者視角的 Google 自然搜尋讀回，補上 Google Search Console 的後台成效資料。

它回答的是：剛剛用無痕 Google 搜到的詞，哪些在 Search Console 近 3 個月也有實際曝光 / 點擊；哪些只是當下前台可見，但後台查詢表還沒有列出；以及目前實際吃曝光的是新頁、首頁，還是舊 PHP URL。

## 來源與限制

- 來源：Google Search Console 已登入 Chrome 匯出。
- 資源：`https://www.observe888.com/`
- 類型：網路搜尋。
- 日期範圍：前 3 個月；圖表範圍顯示 2026-02-18 至 2026-05-17。
- 匯出時間：2026-05-20 16:30 左右，畫面顯示上次更新時間為 3 小時前。
- 原始匯出檔在本機下載與暫存資料夾，僅作分析，不提交進 repo。

限制：

- Search Console 的平均排名是歷史曝光平均，不等於無痕 Google 當下單次搜尋排名。
- Search Console 查詢表會隱藏或省略低量 / 匿名查詢；某個詞前台可見但 GSC 查詢表沒有列出，不代表該詞沒有排名。
- 頁面維度 row sum 不能當成網站總量，因為同一次搜尋可能對多個 URL 形成頁面列。

## 判斷

1. 這次 Search Console 補查確認：核心服務詞不是只有前台單次搜尋可見，後台也有實際曝光，尤其 `台南整骨`、`台南整復`、`台南推拿`、`台南整復推拿`。
2. 目前最大問題不是「沒有搜尋需求」，而是承接頁分散：首頁吃掉大部分曝光，新服務頁 `/services/tainan-tuina/` 尚未進入 Search Console 頁面表。
3. 推薦 / 費用詞有量，但 CTR 偏低；後續應優先微調 FAQ、title / description 與內鏈，不應做假排行或保證改善。
4. 店別、路名、結構定位詞前台表現好，但後台還沒有形成穩定查詢列；適合追蹤，不適合一次大改。
5. 運動按摩、筋膜、按摩、手法詞在 Search Console 的量很小且排名弱，和前台讀回一致：可承接服務邊界，但不是主定位。
6. `結構治療`、`台南結構治療`、`高低肩矯正` 等詞只能作內部觀察與 FAQ 邊界，不應放成 GBP 服務項目或官網主打詞。
7. 舊 PHP URL 仍有 7,215 次頁面維度曝光；技術清理優先級仍高於新增大量薄內容。

## 下一步

1. 1-3 天後重查 Search Console 頁面表，看 `/services/tainan-tuina/` 是否開始出現。
2. 若新服務頁仍沒有進表，再檢查 URL 檢查、索引狀態、sitemap 與內鏈，不先大改文案。
3. 保持 Maps / Google Business Profile 和一般自然搜尋分開追蹤；Search Console 只能補 Google Search 後台資料，不能代表 Maps 排名。
4. 舊 PHP URL 若持續高曝光，下一步仍是主機 / CDN / Cloudflare 或原主機層的 301 規則，而不是只靠 404 JS 導流。
