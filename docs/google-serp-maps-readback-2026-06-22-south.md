# Google Search / Maps 南區讀回 - 2026-06-22

本文記錄北區結束後，只以南區店為主要 Google Search / Maps 目標的公開面抽樣讀回。東區工作室目前不作為 Google Maps 排名目標；本次只確認它仍是官網純預約工作室語境的一部分。

- Search Console：本機最新正式匯出仍是 2026-06-07；2026-06-13 只有等待匯出的提示檔，未取得新 CSV。

## 判斷

1. 南區 Maps 目前不是主要弱點；至少在本輪南區 IP 條件下，南區店已經能吃到泛詞、推薦型詞、南區詞與明興路路名詞。
2. Google Search 的弱點在泛詞與推薦型詞的自然官網結果，尤其 `台南整復推拿` 與 `台南整骨推薦` 沒有穩定進前 8。
3. 6/7 Search Console 已指出 `/services/tainan-tuina/` 尚未進入 GSC 頁面表；今天公開面也顯示服務頁沒有穩定成為泛詞自然結果。
4. 官網不需要大改；比較合理的是補首頁、南區頁、費用頁到 `/services/tainan-tuina/` 的明確內鏈，並更新 sitemap `lastmod`。
