# 見觀自然搜尋每週檢查流程

## 固定時間

每週一次即可。不要每天因單次排名波動修改頁面。

## 1. paid search terms

來源：`02_廣告_AdsControl/backend/data/adscontrol.db`

觀察：

- 哪些搜尋字有 clicks / cost / proxy conversion。
- 哪些字花錢但不產生 LINE / phone proxy。
- 是否出現新的競品品牌字、醫療高風險字、按摩雜訊字。

判斷：

- 高意圖且符合服務：回補 SEO 頁面與 FAQ。
- 雜訊高：先做 Ads 排除或差異說明，不急著做 SEO 頁。
- 競品品牌：追蹤即可，不寫競品品牌頁。

## 2. Search Console

需要登入或授權才能做。

觀察：

- Query：`台南整復`、`台南整復推拿`、`北區整復`、`南區整復`、`台南運動按摩`、`台南整骨推薦`。
- Page：`/services/tainan-tuina/`、`/north/`、`/south/`、`/faq/`、高意圖 notes。
- 指標：impressions、clicks、CTR、average position。

判斷：

- 有曝光無點擊：先改 title / meta description。
- 有點擊但沒有 proxy intent：檢查頁面 CTA、LINE、電話與店別入口。
- 有排名但非目標頁：加內鏈，把意圖導到正確頁。

## 3. 公開 SERP

建議用無痕或乾淨瀏覽器，不登入也可以；不要把單次結果當永久排名。

固定查詢：

- `台南整復`
- `台南整復推拿`
- `台南整復推拿推薦`
- `台南北區整復推拿`
- `台南南區整復推拿`
- `台南運動按摩推薦`
- `台南物理治療 徒手治療`

記錄：

- 前 10 名自然結果。
- local pack 是否有見觀。
- 是否出現第三方推薦頁、Dcard、Gomaji、Wanderlog。
- 見觀有沒有舊網址或錯誤摘要。

## 4. Google Business Profile

需要店主後台或授權才可改。

只讀觀察：

- 店名、地址、電話、網站、預約網址、服務網址是否仍指向 `https://www.observe888.com/`。
- 北區、南區是否混淆。
- 評價裡是否出現可轉成 FAQ 的常見問法。

可改項目需先得到授權：

- 服務項目。
- 商家描述。
- 照片。
- 貼文。
- 網站/預約/服務 URL。

## 5. 內容任務判斷

每週只選 1 到 3 件，不要同時亂開太多頁。

優先順序：

1. 已有曝光、有點擊、符合 LINE/phone proxy 的字。
2. Ads 花費高但 SEO 沒承接的字。
3. 競品明顯有頁、見觀沒有頁的意圖。
4. Search Console 顯示排名 8 到 20 名、可用內鏈或 FAQ 推進的頁。

## 6. 不做的事

- 不做不存在的分店頁。
- 不寫「台南第一」、「最有效」、「保證改善」。
- 不做競品攻擊頁。
- 不把 LINE / phone proxy 寫成已預約、已成交或已到店。
- 不把整復推拿寫成醫療診斷或治療。
