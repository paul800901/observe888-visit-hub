## 核心方向

首頁要從「SEO / AI / 法務導向」改成「人類先看得懂」。

但不是放棄 SEO / AEO / AI：

- `metadata`
- `JSON-LD`
- `llms.txt`
- `sitemap.xml`
- FAQ
- 摺疊區與頁尾

這些地方仍可保留機器可讀資料；只是不要把機器資料裸露成首頁主文案。

## 仍未完成

這次只是第一輪「首頁與文案瘦身」，不是完整前端改版。下一輪仍要補：

1. 真實視覺證據：店內、入口、工作室、操作情境、短影音縮圖。
2. 第一屏可讀性：手機 5 秒內要知道這家在哪、做什麼、多少錢、怎麼約。
3. CTA 去重：不要同時出現兩組功能相同的預約 / 費用 / 導航按鈕。
4. 品牌差異：要讓人看懂見觀不是一般國術館、按摩店或全身亂喬。
5. 雙首頁分工：`/` 負責分流與第一眼理解；`/about/` 負責品牌與理念。
6. SEO / AEO 不刪除，只移到 metadata、schema、FAQ、llms 或摺疊區。

## 不要回退的點

後續改版不可回到：

- 首頁第一屏放「不是醫療診斷」「不宣稱療效」「不保證改善」。
- 用 `結構治療` 當公開主詞。
- 把外部資料來源、Wikidata / Wikipedia / Google Maps 核對目的放成可見主文案。
- 把預約系統後台邏輯寫給客人看。
- 把 `NT$500-800` 當唯一費用主訊號。
- 把首頁寫成症狀關鍵詞清單。

- `llms.txt`、`sitemap.xml` 沒壞。
