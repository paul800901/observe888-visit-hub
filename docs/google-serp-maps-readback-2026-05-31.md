本段補讀使用登入狀態的公開 Maps / Google Search 商家面板，可能受帳號與位置影響；排名只能當作排序因素判斷，不替代無痕排名稽核。

北區對照：

排序因素判斷：

- 南區不是 GBP 資料不足，而是從 `明興路` 放寬到 `南區` / `台南` 後，Google 更看重距離、類別、評論量、評論語意、商家名稱相關性與整體 prominence。
- `明興路整復` / `明興路推拿` 能顯現，代表地址與路名訊號有效；`南區推拿` 不顯現，代表區域泛詞與推拿語意仍弱。
- 南區評論量 `68` 低於北區 `127`，也低於多個泛推拿競品，例如 `誠翊傳統整復推拿` 698、`御手國醫台南會館` 2247、`妙心齋健體整復經絡推拿` 1708、`Relax按摩工作室` 193。
- 南區店名本身沒有 `推拿`，而競品常在商家名稱或公開名稱中直接含 `推拿`、`經絡按摩`、`刮痧拔罐`、`傳統整復推拿`；不建議見觀跟著塞店名關鍵字，但這會影響 Maps 對泛詞的相關性判斷。
- 主類別 `整脊師` 對 `整復` / `整脊` 有幫助；對 `推拿` 詞，Google 可能偏向 `按摩治療師`、`按摩水療`、`運動按摩治療師` 等類別。但南區已有 `按摩師` / `運動按摩師` 作為其他類別，不建議為了 `推拿` 直接改主類別，因為可能傷到整復定位。
- Google Search 商家面板仍可看到舊第三方結果，例如 Workband、小工匠家事網等，且部分外部摘要仍有舊服務詞或醫療化語意；外部資料一致性仍是後續可補強項。

### 實際補救方向修正

1. 不重填 GBP 基本資料；南區目前不是空白或不完整。
2. 優先補真實評論語意：請南區真實顧客自然提到實際分店、明興路、南區、預約制、整復推拿、第一次到店、停車/找路等體驗；不可買評價、不可誘導療效詞。
3. 貼文節奏改成南區在地使用情境，而不是重發泛服務頁：明興路到店、南區預約制、第一次到店流程、停車/入口、服務邊界、費用與時段。
4. 照片不需要短時間大量重複補；現在公開面已看得到最新與業主精選照片。下一步是確認前排照片是否足以辨識入口、空間、路名與工作室預約感。
5. 清理或修正外部舊名錄 / 舊頁摘要，特別是仍在 Google 搜尋結果出現的舊第三方頁，不讓舊醫療化或錯誤服務語意繼續污染公開訊號。
6. 類別不建議直接改主類別；若未來要動，應先做一輪競品類別與排名對照，並接受主類別變動可能同時影響 `整復`、`整脊`、`推拿` 三類詞。
