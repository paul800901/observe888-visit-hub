## 舊網址問題

Search Console 仍列出多個舊 PHP URL，例如：

- `/paper/other_select_index.php?id=3718&title_id=11271&group_id=874`
- `/paper/other_page.php?id=11272`
- `/paper/other_page.php?id=11276`
- `/products/car.php`
- `/album/index.php?title_id=11273`
- `/news/index.php?...`

2026-05-19 live HTTP 讀回：上述高曝光舊 URL 多數回 `404`，目前只靠 `404.html` 的 JavaScript 把真人瀏覽器導去新版入口。這能保護使用者體驗，但不能等同正式 server-side 301。

建議若網域 / 主機 / Cloudflare 可設定轉址，優先做 server-side 301：

| 舊路徑 | 建議新路徑 |
| --- | --- |
| `/paper/other_select_index.php?id=3718&title_id=11271&group_id=874` | `/north/pricing/` |
| `/paper/other_select_index.php?title_id=11271&group_id=874` | `/visit/` |
| `/paper/other_page.php?id=11272` | `/about/` |
| `/paper/other_page.php?id=11276` | `/visit/` 或依舊頁內容確認後改到更準頁 |
| `/products/car.php` | `/services/tainan-tuina/` 或 `/visit/` |
| `/album/*` | `/notes/` |
| `/news/*` | `/notes/` |
| `/paper/promotions_index.php*` | `/visit/` |
