# Google Maps 排名代操判斷與後續策略 - 2026-05-21

## 背景

這份紀錄補上 2026-05-21 對 Google Maps / Google Business Profile 排名代操的判斷。

使用者已確認：過去曾付費購買地圖排序服務，花費數萬元但成效不明顯；後續不應再把「保證地圖排序」代操包當成主要解法。

同時使用者也確認：`observe.boostime.me` 舊站殘留已交由外部公司處理，短期內不列為本專案可控驗收條件。它只作為背景噪音與外部不可控風險記錄，不應在每次 Maps 診斷中反覆當成主要待辦。

## 目前判斷

### 自然搜尋

自然搜尋不是目前最急的主戰場。

依 `docs/maps-tainan-tuina-readback-2026-05-19.md` 與 2026-05-19 使用者無痕讀回，核心服務字如 `台南推拿`、`台南整復`、`台南整復推拿` 已在第一頁前段。自然搜尋後續主要注意更針對性的長尾詞：

- `台南北區整復推拿`
- `台南南區整復推拿`
- `台南整復推拿推薦`
- `台南整骨推薦`
- 費用型、推薦型、地區型與服務邊界型查詢

### Google Maps / 地圖

Google Maps / 地點結果才是目前較麻煩的主戰場。

Google 官方說明本地排名主要由相關性、距離與知名度組合判斷。參考：

- `https://support.google.com/business/answer/7091`
- `https://support.google.com/business/answer/3038177`

見觀目前地圖落後，不應解讀成官網自然搜尋全面失敗，也不應解讀成 PMax 沒效。PMax 只能處理付費曝光與短期詢問，不能直接推升自然 Maps 排名。

## 為什麼很多競品地圖容易出現

競品通常不是靠單一技巧，而是多個 Maps 訊號疊在一起：

1. 店名命中搜尋詞。
   - 很多競品名稱本身含 `推拿`、`整復推拿`、`傳統整復`。
   - Google Maps 對商家名稱仍很敏感。
   - 見觀可以使用真實店名 `見觀結構調理整復所-北區店` / `見觀結構調理整復所-南區店`，但不應把店名欄改成 `台南推拿推薦`、`台南整復推拿`、`整骨按摩` 等關鍵字堆疊版本。

2. 主類別更貼近搜尋意圖。
   - 見觀目前主類別為 `整脊師`。
   - 前排競品常見類別較接近 `按摩治療師`、`運動按摩治療師`、`整骨院`。
   - 是否調整主類別是高影響決策，需等待 2026-05-19 服務項目 / 描述與 2026-05-20 相片補強公開面收斂後，再用乾淨 Maps 讀回判斷。

3. 評論量與評論語言差距。
   - 既有基準中，北區約 126 則評論，南區約 68 則評論。
   - 前排競品可見 400 則以上甚至 1000 則以上評論。
   - 評論中若自然出現 `推拿`、`整復`、`按摩`、`肩頸`、`腰`、`調理` 等實際服務語言，會同時補相關性與知名度訊號。

4. 距離與搜尋位置。
   - Maps 比自然搜尋更吃使用者位置。
   - 同一個查詢，在北區、南區、永康、東區、中西區看到的地點排序可能不同。
   - 後續排名驗收必須固定查詢條件與位置，不可用單次個人手機結果直接下結論。

5. GBP 新訊號需要時間。
   - 2026-05-19 才完成服務項目與商家描述第一輪修正。
   - 2026-05-20 才完成南北店相片補強。
   - 公開 Maps 排名、相片排序與商家面板顯示可能有審核與快取延遲。

## 對地圖排序代操公司的判斷

地圖排序公司通常會做：

- 填 GBP 類別、服務、描述、照片、貼文。
- 做外部商家頁或名錄，補名稱、地址、電話一致性。
- 寫地方關鍵字頁或部落格。
- 定期上傳相片 / 貼文。
- 給排名報表。

這些工作不是完全沒用，但只能補一部分相關性與知名度；無法保證 Maps 排名。若服務商沒有詢問或盤點以下資料，就很可能只是模板代操：

- 南北店實際店名與招牌 / 官方名稱證明。
- 主類別與副類別策略。
- 服務邊界與不能宣稱的醫療 / 治療 / 療效用語。
- 南北店地址、營業型態、預約方式與客群差異。
- 實際來客來源、LINE / 電話 / 導航互動。
- 評論邀請流程與真實服務語言。
- 競品類別、評論量、店名命中、距離與公開面差異。
- 固定條件的 Maps 排名讀回方法。

若服務商承諾保證排名，但不處理上述內容，預期成效有限。若服務商依靠假評論、店名堆關鍵字、假點擊、假導航或其他灰色操作，短期可能有波動，但有商家檔案風險，不列為見觀策略。

## 見觀後續策略

1. 先等 2026-05-19 GBP 服務項目 / 描述與 2026-05-20 相片補強公開面收斂，不在審核 / 快取期間連續亂改。
2. 建立乾淨 Maps 排名表：無痕、固定查詢詞、固定位置、分北區 / 南區記錄。
3. 1-2 週後若仍未改善，再評估主類別是否需要測試；不要先改店名，不要關鍵字堆疊。
4. 建立真實評論邀請流程，不誘導、不買評論，但可提醒客人描述實際服務體驗與到店原因。
5. 持續補 Google Business 相片：門面、入口、停車、費用、預約、服務情境、第一次到店流程。
6. 保持官網、GBP、FAQ、服務頁文字一致，避免一邊叫整脊、一邊叫整復、一邊叫結構調理到 Google 無法歸類。
7. Maps 排名與 PMax 分開看：PMax 觀察 LINE / 電話 / 現場來客；Maps 觀察自然地點排序、商家面板點擊、導航、電話與評論。

## 不做事項

- 不買「保證地圖排名」代操包作為預設方案。
- 不把店名欄改成關鍵字堆疊版本。
- 不買假評論，不做假導航、假點擊或其他灰色操作。
- 不把 `observe.boostime.me` 舊站殘留列為短期內部可控驗收條件；它只保留為外部不可控背景風險。
- 沒有乾淨公開 Maps 讀回前，不宣稱 Maps 排名已改善。
