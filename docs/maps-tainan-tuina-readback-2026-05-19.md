## 見觀目前可能弱點

| 面向 | 現況 | 風險 |
| --- | --- | --- |
| Maps 類別 | 主要常顯示 `脊醫` / `整脊師` | 對 `台南推拿`、`台南整復推拿` 可能不如按摩 / 運動按摩 / 整骨院貼近 |
| 評論量 | 北區約 126，南區約 68 | 與 1707、436、226、222 這類前排競品有差距 |
| 商家名稱 | 見觀品牌名清楚，但不直接含 `推拿` | 不建議硬塞名稱；但 Google 對名稱命中可能有偏好 |
| 服務項目 | 後台已讀到 `結構評估`、`動作與姿勢觀察`、`徒手調理` | 對 `推拿` 字的明確性可能不足 |
| 商家貼文 | 已有短漫貼文，服務頁導流貼文待評估 | Maps 需要更明確的服務語言與新鮮內容 |
| 照片 | 南北店都有照片，但缺本輪公開排序中的對照 | 競品可能用環境、老師、服務圖提升信任 |
| 地理位置 | 查詢結果受位置影響，永康 / 東區競品很多 | 見觀北區 / 南區需要各自強化 local relevance |
