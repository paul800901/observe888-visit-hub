資料刪除說明

如果你曾透過見觀結構官方網站、LINE、表單、電話或社群工具提供資料，可以依本頁方式提出查詢、修正或刪除請求。

最後更新：2026-06-16

如何提出刪除請求

  1. 請寄信到 observe88888@gmail.com。
  2. 信件主旨請寫「資料刪除請求」。
  3. 請提供你希望刪除的資料類型，例如 LINE 預約聯絡資料、表單資料、網站聯絡資料，或 Meta / Threads / Instagram 工具測試相關資料。
  4. 為避免誤刪他人資料，我們可能會請你補充可識別該筆資料的必要資訊。

處理方式

收到請求後，我們會確認資料來源與保存位置，刪除不再需要保存的聯絡資料或社群工具測試資料。

若資料涉及已完成的預約、內部紀錄、法規、帳務或平台安全要求，可能需要保留必要紀錄；這種情況會盡量降低可識別資訊，只保留必要範圍。

Meta / Threads / Instagram 工具

見觀結構的 Meta / Threads / Instagram 工具主要用於讀取自己帳號的貼文與洞察報告。若你希望撤回平台授權，亦可至 Meta、Facebook、Instagram 或 Threads 的帳號設定中移除相關應用程式連結。
